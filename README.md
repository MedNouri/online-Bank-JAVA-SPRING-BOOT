# online-Bank
A CRUD school project  :Online bank Application made with java spring Boot

# Features
• creat a user account and create multiple  bank account 
• delete 
• view transaction history 
• ask chequebooks and debit credit 


###  Frameworks:
• Spring Core
• Spring MVC 
• Hibernate 
• Bootstrap
